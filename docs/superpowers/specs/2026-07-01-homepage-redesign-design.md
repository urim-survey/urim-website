# 횡성우림측량 홈페이지 전체 재설계 — 설계 문서

날짜: 2026-07-01

## 배경 / 목표

기존 Next.js 프로젝트(7페이지: `/`, `/about`, `/services`, `/guide`, `/blog`, `/faq`, `/contact`)를 참고 사이트 [siwolarchi.com](https://www.siwolarchi.com/)(건축사사무소, 미니멀 모노톤 포트폴리오 스타일)를 벤치마킹해 처음부터 전면 재설계한다. 이번 스코프는 **비주얼/구조 재설계만** — Supabase 연동, 실사진 업로드는 별도 후속 작업.

## 참고 사이트 분석 요약

- 헤더: 플랫 가로 네비 (Home/Portfolio/About/Contact/Blog)
- 히어로: 풀블리드 배경 이미지
- 본문 흐름: Featured Projects 3개 → Recent News 3개 → Contact
- 컬러: 모노톤(흑백) 미니멀
- 이미지: 풀블리드 히어로 + 3열 정사각형 갤러리 그리드
- 톤: 전문적·미니멀, 이미지 중심

## 확정 사항

### 1. 정보구조 / 네비게이션

- URL 구조 유지: `/`, `/about`, `/services`, `/guide`, `/blog`, `/faq`, `/contact` (7개, CLAUDE.md 원안 그대로)
- 헤더: 텍스트 로고("횡성우림측량") + 7개 메뉴 플랫 배치 + 전화번호(033-345-1253) 우측 고정
- 모바일: 햄버거 → 풀스크린 오버레이 네비
- 푸터: 업체명/주소/전화/영업시간 + 페이지 링크 + 카카오채널 버튼(링크 미정, `href="#"` placeholder)

### 2. 컬러 / 타이포그래피

완전 모노톤 (참고 사이트와 동일, 포인트 컬러 없음). CLAUDE.md에 기존 명시된 네이비(#003087/#0057b8) 스펙과 다름 — 이 설계 확정 후 CLAUDE.md 디자인 방향 섹션 갱신 필요.

- 배경: `#FFFFFF`
- 텍스트 기본: `#111111`
- 텍스트 보조: `#6B6B6B`
- 보더/구분선: `#E5E5E5`
- 호버/액티브: `#333333` (명도 변화만, 색상 변화 없음)

폰트: **Pretendard** (CLAUDE.md의 Noto Sans KR에서 변경)
- 히어로 타이틀: 48~64px, bold
- 섹션 타이틀: 28~36px, semibold
- 본문: 16px, regular, line-height 1.7
- 영문 라벨(`eng-label`, 기존 코드 계승): 12px, uppercase, letter-spacing 넓게

### 4. 공통 컴포넌트

- `Header` (네비 + 모바일 햄버거 오버레이)
- `Footer`
- `HeroSection` (풀블리드 이미지 + 오버레이 텍스트, 페이지 공용)
- `ServiceCard`
- `BlogCard`
- `StepTimeline` (`/guide` 전용)
- `FaqAccordion`
- `ContactForm` (기존 로직 유지, 스타일만 교체)
- `Button` (primary / text-link 2종)
- `FadeInSection` (Framer Motion 래퍼, 공용)

### 5. 애니메이션 (Framer Motion)

- `framer-motion` 패키지 신규 추가
- 섹션/카드: `whileInView`로 opacity 0→1, y 20→0, duration 0.6
- 카드 그리드: `staggerChildren`으로 순차 등장
- 히어로: 페이지 로드시 1회 페이드인 (스크롤 트리거 아님)
- 패럴랙스·3D 등 과한 효과 제외, "미묘한 페이드인" 수준까지만
