#### 블로그 상세 페이지 개편 (`/blog/[slug]`)
- 제목 맨 위·가운데 정렬 → 썸네일(블러 효과) → 본문 순서로 재배치
- 하단 **다른 글 보기** 최신 3개 + **목록으로** 버튼 추가

- 블로그 상세 404 오류: 한글 슬러그 URL 인코딩 문제 → `decodeURIComponent` 적용
- 새 글 슬러그 영문+타임스탬프 방식으로 변경 (한글 URL 재발 방지)
- 히어로 이미지 검정 깜빡임: CSS background → `<Image priority>` 교체

#### 배포 환경 정상화
- Vercel이 `main` 브랜치 배포 중인데 `master`에만 푸시하던 문제 발견·수정
- **앞으로 푸시 시 `master`와 `main` 동시 업데이트**

#### 작업 운영 개선
