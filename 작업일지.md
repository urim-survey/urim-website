### 2026-07-04 (오늘)

**주요 완성 기능:**

#### 1. 블로그 글쓰기 시스템 구축
- `/blog/write` 페이지 신규 생성
- TipTap 리치텍스트 에디터 적용
- 글 작성 후 "발행하기" 클릭 → Supabase DB(`posts` 테이블)에 저장
- 블로그 목록(`/blog`)에서 작성된 글 카드로 표시

#### 2. 에디터 기능 상세
| 기능 | 내용 |
|------|------|
| 폰트 | Pretendard 포함 한글 폰트 **24종** (구글 폰트) |
| 폰트 크기 | 8px ~ 72px 19단계 선택 |
| 텍스트 | 굵게·기울이기·밑줄·취소선 |
| 정렬 | 왼쪽·가운데·오른쪽 |
| 색상 | 기본 색상 7개 + 직접 선택 |
| 목록 | 글머리·번호 목록 |
| 이미지 | 컴퓨터에서 업로드 → Supabase Storage 저장 |
| 이미지 크기 | 클릭 후 양쪽 핸들 드래그로 조절 |
| 이미지 정렬 | 클릭 후 왼쪽·가운데·오른쪽 버튼 선택 |
| 링크·인용·코드 | 지원 |

#### 3. 대표(썸네일) 이미지
- 글쓰기 페이지에서 컴퓨터 이미지 선택 → 미리보기 표시
- **이미지 변경** / **이미지 제거** 버튼 제공
- Supabase Storage `blog-images` 버킷에 자동 업로드

#### 4. 블로그 상세 페이지 개편 (`/blog/[slug]`)
- 제목을 **맨 위·가운데 정렬**로 변경
- 썸네일 이미지 → **살짝 뿌옇게** 블러 효과 적용
- 하단에 **다른 글 보기** (최신 3개 카드) 추가
- **목록으로** 버튼 추가

#### 5. 버그 수정
- 블로그 상세 페이지 **404 오류** 수정
  - 원인: 한글 슬러그 URL 인코딩 문제 (`블로그-테스트-1` → `%EB%...`)
  - 해결: `decodeURIComponent` 적용 + 새 글 슬러그를 영문+타임스탬프로 변경
- 홈 히어로 이미지 **검정 깜빡임** 수정
  - 원인: CSS background-image 지연 로딩
  - 해결: Next.js `<Image priority>` 태그로 교체

#### 6. 기술 연동 완료
- Supabase DB (`posts` 테이블) 연동 완료
- Supabase Storage (`blog-images` 버킷) 연동 완료
- RLS(보안 정책) 설정 완료 → 이미지 업로드/조회 허용
